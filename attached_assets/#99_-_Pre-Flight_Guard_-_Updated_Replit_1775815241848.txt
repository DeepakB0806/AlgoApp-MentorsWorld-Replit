#99 - Task #99 — Pre-Flight Guard: Expired Contract Terminal Close
What & Why
The Error Command Centre (Task #98) routes Kotak API errors through handleBrokerError(), but closeTrade() has two early-exit paths that bypass it entirely. When an options contract has expired, its token disappears from brokerSymbolToTokenMap. The function detects this locally and writes close_failed — never reaching the broker or the switchboard. The persistent retry loops (startPersistentExit, startPersistentSquareOff) then pick up those close_failed trades and replay the same dead path forever — a zombie loop that continues until the server restarts. The same problem occurs when the live quote fetch fails for an option leg.

Done looks like
When closeTrade() detects an option symbol is missing from the Scrip Master, it immediately marks the trade as closed (pnl=0), writes a PFL entry, and returns — no retry, no zombie loop.
When the live quote fetch fails for an option, the same terminal-close treatment applies instead of close_failed.
Two new seed patterns added to error_routing as defence-in-depth.
The "Stuck Exits" Force-Close section never appears for expired contracts — the system self-heals in one pass.
Out of scope
Changes to startPersistentExit or startPersistentSquareOff loop logic.
Changes to handleBrokerError() or the error routing switchboard.
Any UI changes.
Tasks
Task 1 — Pre-flight token guard in closeTrade()
Inside the Kotak branch of closeTrade(), immediately after the basic symbol/exchange/productType validation block (line 1117) and before the ctPriceMode check (line 1119), inject this exact block:

// PRE-FLIGHT DEAD CONTRACT GUARD
if (isOptionExchange(trade.exchange)) {
  const preflightToken = brokerSymbolToTokenMap.get(trade.tradingSymbol);
  if (!preflightToken) {
    const msg = `[TE] Pre-flight Abort: Contract ${trade.tradingSymbol} missing from Scrip Master (expired).`;
    console.warn(msg);
    addProcessFlowLog({
      planId: trade.planId,
      planName: "Auto-Close Guard",
      signalType: "SYSTEM",
      alert: "EXPIRED_CONTRACT",
      resolvedAction: exitAction,
      blockType: "SQUARE_OFF",
      actionTaken: "force_closed",
      message: msg,
      broker,
    });
    const closedUpdate = await storage.updateStrategyTrade(trade.id, {
      status: "closed",
      pnl: 0,
      exitedAt: now,
      updatedAt: now,
      exitPrice: trade.ltp || trade.price || 0,
      exitAction,
    });
    tradingCache.invalidateOpenTrades(trade.planId);
    return closedUpdate || trade;
  }
}
