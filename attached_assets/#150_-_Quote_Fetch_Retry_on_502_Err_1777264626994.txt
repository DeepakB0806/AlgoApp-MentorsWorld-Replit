#150 - Quote Fetch Retry on 502 Errors

What & Why
When placing a LMT order, the Trading Engine calls Kotak's quote REST API to get the live LTP before computing the limit price. If Kotak's API returns a transient 502 Bad Gateway (confirmed in production PFL on 2026-04-27 at 09:33 IST for NIFTY2650524300CE), the TE immediately ABORTs the trade — even though the failure was momentary and a retry would likely succeed.

The 09:15–09:30 IST window at market open is the highest-risk period for this: Kotak's servers are under peak load and transiently return 502s. This is exactly when most entry signals fire.

No new user setting is needed. A hardcoded retry (3 attempts, 1 s apart) is the right fix — the interval is short enough to be transparent, long enough for the broker's servers to recover from a transient overload.

Done looks like
getQuote in the Execution Layer retries up to 3 times with a 1 s delay when it receives a 502 or any error response that contains no LTP
Each retry attempt is logged clearly: [EL] Quote retry attempt 2/3 for NIFTY2650524300CE (previous error: API error: 502 Bad Gateway)
Only after all 3 attempts fail does the TE log ABORT and halt the order
No new UI fields or DB settings required
Out of scope
Retry for order placement failures (covered by existing settings)
Retry for non-transient errors (e.g. invalid token, 401 — these should still fail immediately)
Steps
Add retry loop to getQuote — In el-kotak-neo-v3.ts, wrap the quote fetch and LTP parse in a retry loop (max 3 attempts, 1 s delay between attempts). On each failure, log the attempt number and the error. Only return the failure result after all attempts are exhausted. Do not retry on auth errors (401/403) — only on 5xx/network/no-LTP responses.

async getQuote(
    config: BrokerConfig,
    exchange: string,
    token: string,
  ): Promise<{ success: boolean; ltp?: number; error?: string }> {
    if (!this.ready && !(await this.ensureReady())) {
      return { success: false, error: `EL not ready: ${this.initError || "initialization failed"}` };
    }

    const endpoint = this.getEndpoint("quotes");
    if (!endpoint) return { success: false, error: "quotes endpoint not configured" };

    const headers = this.buildHeadersForEndpoint(endpoint, {
      accessToken: config.accessToken,
      sessionId: config.sessionId,
      consumerKey: config.consumerKey,
    });
    const url = `${config.baseUrl}/script-details/1.0/quotes/neosymbol/${exchange}|${token}/all`;

    // TASK #150: 3-Attempt Retry Loop for Transient 502 Bad Gateway
    let lastError = "";
    for (let attempt = 1; attempt <= 3; attempt++) {
      try {
        const res = await fetch(url, { method: "GET", headers });
        
        // Bail out immediately on Auth errors (do not retry 401/403)
        if (res.status === 401 || res.status === 403) {
           return { success: false, error: `API auth error: ${res.status} ${res.statusText}` };
        }

        if (!res.ok) {
          throw new Error(`API error: ${res.status} ${res.statusText}`);
        }

        const data = await res.json();
        const ltp = data?.data?.[0]?.ltp ?? data?.ltp;

        if (ltp !== undefined && ltp !== null) {
          return { success: true, ltp: Number(ltp) };
        } else {
          throw new Error("LTP missing from valid response payload");
        }
      } catch (err: any) {
        lastError = err.message;
        if (attempt < 3) {
          console.log(`${LOG_PREFIX} Quote retry attempt ${attempt}/3 for ${exchange}|${token} (previous error: ${lastError})`);
          await new Promise((r) => setTimeout(r, 1000)); // 1-second backoff
        }
      }
    }

    return { success: false, error: `Quote fetch failed after 3 attempts. Last error: ${lastError}` };
  }
