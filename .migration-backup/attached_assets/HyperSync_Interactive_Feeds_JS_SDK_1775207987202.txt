HyperSync Interactive Feeds JS SDK
HS JS SDK connects to HyperSync back-end for order updates.
CONNECTION
CONNECTION - 1. Create WS Object
This shows how to make connection with Interactive data js-api, Here server_url specifies the URL to which to connect.

websocket
HSIWebSocket
Sample
var hsiSocket = new HSIWebSocket(server_url);

JS API EVENTS
JS API EVENTS - 1. On Open
This event occurs when socket connection is established.

websocket
HSIWebSocket.onopen
Sample
hsiSocket.onopen = function () 
{
	//code
}
JS API EVENTS - 2. On Message
This event occurs when client receives data from server.

websocket
HSIWebSocket.onmessage
Sample
hsiSocket.onmessage = function (message) 
{
	//message -> connection response/order update/trade update/position update
}
JS API EVENTS - 3. On Close
This event occurs when connection is closed.

websocket
HSIWebSocket.onclose
Sample
hsiSocket.onclose = function () 
{
	//code
}
JS API EVENTS - 4. On Error
This event occurs when there is any error in communication.

websocket
HSIWebSocket.onerror
Sample
hsiSocket.onerror = function () 
{
	//code
}
JS API METHODS
JS API METHODS - 1. Send
This function is used for sending the json request.

websocket
HSIWebSocket.send
Sample
hsiWebSocket.send(request);
Request
{
    "type": "cn",
    "x-access-token": "sx-access-token1",
    "source": "WEB"
}
JS API METHODS - 2. Close
This function is used for closing the webscoket connection.

websocket
HSIWebSocket.close
Sample
hsiWebSocket.close();
